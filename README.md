# CYBER_START – Página de defacement

Este repositorio ahora se limita a mostrar una única página tipo *defacement*.

## Cómo ejecutar la página
1. Instala las dependencias del front-end y genera la versión estática:
   ```bash
   cd PaginaSemaforos/PaginaSemaforos
   npm install
   npm run build
   ```
2. Sirve la carpeta `dist` resultante con el servidor estático de tu preferencia (por ejemplo `npm run preview`, `npx serve dist`, o cualquier hosting).
3. Abre el navegador en la URL donde publiques el contenido y verás directamente la pantalla de defacement.

Para un vistazo rápido durante el desarrollo puedes usar `npm run dev` dentro de `PaginaSemaforos/PaginaSemaforos`.
